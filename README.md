## 📦 快速开始

### 方式一：直接使用（推荐）
1. 打开 [在线演示](https://ASJ-Alita.github.io/resume-builder)
2. 选择心仪的模板
3. 填写简历信息
4. 导出PDF或生成分享链接

### 方式二：本地运行
```bash
# 克隆仓库
git clone https://github.com/yourusername/resume-builder.git

# 进入目录
cd resume-builder

# 直接打开index.html
open index.html
# 或使用Python启动服务器
python -m http.server 8080
```

### 方式三：部署到GitHub Pages
1. Fork这个仓库
2. 进入 Settings → Pages
3. 选择 Source: main branch
4. 等待部署完成（约2分钟）
5. 访问 `https://yourusername.github.io/resume-builder`

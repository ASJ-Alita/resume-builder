# resume-builder

🚀 AI Resume Builder - 智能简历生成器

用AI和Vibe Coding打造的现代简历生成器，支持多模板、实时预览、AI辅助优化

[在线演示](https://asj-alita.github.io/resume-builder)

## 🎯 功能特色

- 🎨 **5套精选模板** - 经典商务、简约清新、技术极客、创意活力、学术专业
- 🤖 **AI智能优化** - 输入职位，AI帮你生成专业的简历内容
- 📱 **响应式设计** - 完美适配电脑、手机、平板
- 🖨️ **多格式导出** - PDF打印、JSON备份、图片分享
- 🌙 **暗色模式** - 保护眼睛的专业暗色主题
- 🔗 **一键分享** - 生成分享链接，简历随身带
- ⚡ **实时预览** - 编辑即所见，告别排版烦恼
- 💾 **自动保存** - localStorage本地存储，数据不丢失

## 🚀 快速开始

### 直接使用
1. 打开 [在线演示](https://asj-alita.github.io/resume-builder)
2. 选择心仪的模板
3. 填写简历信息
4. 导出PDF或生成分享链接

### 本地运行
```bash
# 克隆仓库
git clone https://github.com/ASJ-Alita/resume-builder.git
cd resume-builder

# 直接打开
open index.html
```

### 部署到GitHub Pages
1. Fork 这个仓库
2. 进入 Settings → Pages
3. 选择 Source: main branch
4. 等待部署完成（约2分钟）
5. 访问 `https://你的用户名.github.io/resume-builder`

## 🛠️ 技术栈

- HTML5 + CSS3 + JavaScript ES6+
- Google Fonts (Noto Sans SC, JetBrains Mono)
- Lucide Icons
- localStorage 本地存储

> 💡 无需任何框架，纯原生实现，加载速度飞快！
